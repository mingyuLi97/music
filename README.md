# music

## 仿网易云音乐移动端

> 此项目仅供学习和交流

### 在线演示

> 可直接用自己网易云账号登录，承诺不保存窃取任何关于您的信息。

[wangyiyun-music](http://121.40.162.237:8003/)

## 技术栈

- vue
- Vue-cli3
- vuex
- vue-router
- vue-lazyload
- vant
- axios

## 实现功能

- 推荐页面
  - 广告轮播
  - 歌单推荐
  - 新歌推荐
  - 热门歌手
- 排行榜
  - 歌曲排行榜
  - 歌手排行榜
- 歌单
  - 歌单分类
  - 歌单详情
- 个人信息（我的）
  - 基础信息
  - 听歌排行
- 播放页面
  - 全局播放控制条
  - 歌词滚动显示
  - 音量 / 播放进度控制
  - 循环模式 / 切歌
  - 编辑播放列表
